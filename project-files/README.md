# Project Files Guide

Use this folder to store either full PLC projects or review-friendly exports for companies that cannot open proprietary files.

## Recommended contents
- Platform source files (optional):
  - `Siemens-TIA/`
  - `Rockwell-RSLogix/`
  - `CODESYS/`
- Review package exports (recommended):
  - `exports/ladder_logic.pdf`
  - `exports/tag_list.csv`
  - `exports/io_list.xlsx`
  - `exports/hmi_screens.pdf`

## Why this looks professional
- Easy for reviewers to inspect logic and standards without needing your exact PLC software versions.
- Better for procurement, maintenance, and technical audits.
- Keeps proprietary source archives optional while preserving engineering evidence.
